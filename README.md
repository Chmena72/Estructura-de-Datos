# Sistema de Tabla Hash – Gestión de Inventario

Implementación en Python de una **tabla hash con encadenamiento separado** para gestionar productos en inventario. Incluye **pruebas de rendimiento**, **visualización de estadísticas**, **carga desde CSV**, y **generación de gráficas interactivas**.

---

## ¿Qué es esto?

Este proyecto simula un sistema de inventario que usa una **tabla hash** para almacenar productos. Permite:

- Insertar, buscar, actualizar y eliminar productos.
- Cargar productos desde un archivo `.csv`.
- Ejecutar pruebas de rendimiento con diferentes tamaños de datos.
- Generar **tablas y gráficas** de rendimiento (tiempos de inserción, búsqueda, eliminación y colisiones).

---

## Tecnologías usadas

- **Python 3.x**
- `matplotlib` – para generar gráficas
- `pandas` – para análisis de datos
- `csv` – para leer archivos CSV
- `json` – para guardar resultados de pruebas
- `time` – para medir tiempos de ejecución
